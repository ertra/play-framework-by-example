Play Framework in Java by example

Query parameter examples:
1. /urlWithoutParameter - Normal URL without parameter
2. /urlWithoutParameter/new - Nested URL
3. /param1?name=Tomas - will write "Hello Tomas !"
4. /param2?name=Jack - will write "Hello Jack !" as a html page
5. /param3/12345 - will write "ok: 12345" , get part of the URL, this time with the product Id
6. /param4/tex1/text2/text3 - will write "URL : tex1/text2/text3"

Static files:
1. /assets/images/play-logo.png - showing a Play logo

Templating:
1. /showProducts - ArrayList iteration example
2. /tagExample - example of usage of tag in view
3. /templateExample - template example in the view

Cookies:
1. /setCookie - set the Cookie
2. /readCookie - read the Cookie
3. /deleteCookie - delete the Cookie

Sessions:
1. /login - put user into the Session
2. /user - read the Session
3. /logout - delete the user form the Session

Different outputs:
1. /json1 - JSON output

reCaptcha Example: (Play Framework Java reCaptcha implementation example)
1. /recaptcha - usage of reCaptcha

Database Example:
1. /insertBook?name=Book1&author=John - insert a Book into the DB
2. /insertBook?Another_book&author=Tomas - insert a Book into the DB
3. /getBooks - display all books

OAuth Examples:
1. /toFacebookOAuth - redirect to Facebook for OAuth
2. /toGoogleOAuth - redirect to Google for OAuth (to try Google OAuth on Heroku, you need to be part of this group https://groups.google.com/forum/#!forum/risky-access-by-unreviewed-apps (more info https://support.google.com/code/contact/oauth_app_verification))

Async Examples:
1. /async - Asynchronous example
