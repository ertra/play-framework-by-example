Query parameter examples URLExamplesController.java
1. /urlWithoutParameter - Normal URL without parameter
2. /urlWithoutParameter/new - Nested URL
3. /param1?name=Tomas - will write "Hello Tomas !"
4. /param2?name=Jack - will write "Hello Jack !" as a html page
5. /param3/12345 - will write "ok: 12345" , get part of the URL, this time with the product Id
6. /param4/tex1/text2/text3?param1=ParamExample - will write "URL wih multiple '/': tex1/text2/text3 and one query parameter: ParamExample"
7. /param5?item=1&&item=2&&item=3 - URL wih list: [1, 2, 3]

Static files:
1. /assets/images/play-logo.png - showing a Play logo

Templating:
1. /showProducts - ArrayList iteration example
2. /tagExample - example of usage of tag in view
3. /templateExample - template example in the view

Cookies examples CookieExampleController.java
1. /setCookie - set the Cookie named 'aaa' to value 'bbb'
2. /readCookie - read the value of Cookie named 'aaa'
3. /deleteCookie - delete the Cookie names 'aaa'

Sessions examples SessionExampleController.java
1. /login - put user into the Session
2. /user - read the Session and display it
3. /userLoginCheck - example of Secured.class, will redirect to '/', if no session is present
4. /logout - delete the user form the Session
